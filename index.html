HTML/CSS Cheat Sheet

Paragraph HTML

Code Explantion
h1-h6 Ranks header size by numbers, great for sub paragrapghs
p Used to make paragraphs.
ul, ol ul is used to make unordered list and ol makes an ordered list.
li Creates the entires in a list.

Media HTML

Code Explantion
vid Adds video to HTML
img Adds image to HTML
a Turns plain text to a url, use target=_blank open in new tab.
audio Adds audio tracks to HTML, use control tag to add controls.

Creating Tables

Code Explantion
table Create's a data table header
tr Creates a new row.
th Creates a table header, best for starting a new row.
td Describes a plain table cell.
colspan Determines how far a column can go in a grid.
rowspan Determines how far a row can go in a grid.

CSS Box Layout

Code Explantion
margin Clears an area outside the border. The margin is transparent.
border A border that goes around the padding and content.
padding Clears an area around the content. The padding is transparent.

CSS

Code Explantion
display: grid Turns an area into a grid
grid-template-columns Specifies the size of a column.
grid-template-rows Specifies the size of a row.
align-content Vertucally aligns content.
justify-content Align the grid inside the table.
